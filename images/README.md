# 图片资源说明

## 登录页面所需图片

请准备以下图片资源：

1. **logo.png** (160x160)
   - 应用 Logo
   - 建议使用圆形或圆角方形设计
   - 背景透明或白色

2. **phone.png** (40x40)
   - 手机图标
   - 用于登录按钮
   - 建议使用绿色主题色 #00BFA5

3. **travel-bg.jpg** (750x500)
   - 首页背景图
   - 建议使用旅行、风景相关图片

## 临时解决方案

如果暂时没有图片资源，可以：
1. 暂时注释掉 image 标签
2. 或使用 emoji/文字替代图标
3. 使用在线图片服务（需要配置合法域名）
